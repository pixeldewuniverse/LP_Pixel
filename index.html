🏗 Landing Page System + CRM Database

  • Landing page multi-slug /lp/[campaign] — mobile-first
  • Fixed template: Hero → Benefits → Testimonial → CTA → FAQ
  • Konten dikelola dari Payload CMS (tanpa re-deploy)
  • Database CRM: tabel leads, customers, transactions
  • Form capture dengan validasi phone format internasional (+62)
  • Auto-sync leads → customers (INSERT / UPDATE)
  • Admin dashboard CRM: filter, tag, export CSV/JSON LAL-ready
  • Auth system: admin role + protected routes
  • SEO foundation: sitemap.xml, robots.txt, JSON-LD, OG tags
  • Email konfirmasi otomatis via Resend (3 bahasa)

📡 Pixel & Tracking Setup (3 Platform)

  • Meta Pixel — PageView, ViewContent, Lead event
  • Google Tag Manager container + GA4 property setup
  • TikTok Pixel — PageView, Lead event
  • dataLayer push untuk semua event form submission
  • Verifikasi via Meta Events Manager + GTM Preview Mode
  • Dokumentasi event tracking untuk tim ads
